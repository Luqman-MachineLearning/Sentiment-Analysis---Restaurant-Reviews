

# Sentiment Analysis of Restaurant Reviews

## 📌 Project Overview
This project focuses on building a machine learning model to classify restaurant reviews as either **positive or negative**. The goal is to understand customer sentiment using natural language processing (NLP) techniques.

The dataset contains textual reviews labeled based on sentiment, and the model learns patterns from these reviews to predict sentiment for unseen data.

---
## 📁 Files

- 📊 [People who Reviewed](./Restaurant_Reviews.tsv)
- 📓 [full Notebook code](./Sentiment%20Analysis%20of%20Restaurant%20Reviews.ipynb)


---

## 📊 Dataset
- **File:** `Restaurant_Reviews.tsv`
- **Description:** Contains restaurant reviews with sentiment labels.
- **Columns:**
  - `Review` → Text of the customer review
  - `Liked` → Sentiment label (1 = Positive, 0 = Negative)

2. Text vectorization (Bag of Words / CountVectorizer)
3. Model training (Naive Bayes or similar classifier)
